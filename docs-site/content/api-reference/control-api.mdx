# Control API

Base URL (local): `http://localhost:3000`

Use the control API from your backend services, not directly from game clients.

## Main Endpoints

- `POST /projects`
- `GET /projects`
- `POST /projects/:id/keys`
- `GET /projects/:id/keys`
- `POST /projects/:id/keys/:keyId/revoke`
- `POST /projects/:id/keys/:keyId/rotate`
- `POST /tokens`
- `GET /metrics`

## Internal Endpoint

- `GET /internal/key-status`

This endpoint is intended for data-plane internal validation (revocation/scope checks).

## Token Claims

Nexis client tokens include:

- `project_id`
- `issued_at`
- `expires_at`
- optional `key_id`
- optional `aud`

Keep token TTL short for gameplay sessions.
